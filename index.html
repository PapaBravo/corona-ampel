Berlins Corona-Ampel

Hier ist eine kleine Visualisierung der Corona-Ampel von Berlin. Die Daten sind gesammelt im Open Data Projekt berlin_corona_cases.

Der Code für diese Seite ist auf Github zu finden. Pullrequests sind willkommen!

Ampelfarben beim R-Wert:
R-Wert < 1,1 = Grün
R-Wert mindestens 3 Mal in Folge ≥ 1,1 = Gelb
R-Wert mindestens 3 Mal in Folge ≥ 1,2 = Rot

Ampelfarben bei Neuinfektionen:
Zahl < 20 je 100.000 Einwohner*innen = Grün
Zahl ≥ 20 je 100.000 Einwohner*innen = Gelb
Zahl ≥ 30 je 100.000 Einwohner*innen = Rot

Ampelfarben bei Auslastung der Intensivstationbetten:
Anteil < 15 % = Grün
Anteil ≥ 15 % = Gelb
Anteil ≥ 25 % = Rot
